VPS Connection Status Demo

How to view the ConnectionStatusExample component:

  1. Add the route to your router configuration (already done)
  2. Navigate to /connection-status in your browser when your app is running

Component Usage

The ConnectionStatusIndicator component can be used in any of your Vue components like this:

<template>
  <ConnectionStatusIndicator :showDetails="true" />
</template>

<script setup>
import ConnectionStatusIndicator from './components/ConnectionStatusIndicator.vue';
</script>

Component Features

  • Shows connection status (connected/disconnected)
  • Displays the connection source (VPS)
  • Indicates when data is being received with a pulsing animation
  • Shows last update time and message count
  • Displays any connection errors

How to Test

You can test the VPS connection status using the test script:

node scripts/test-vps-connection-status.js

Note: This example uses a mock RelayConnectionBridge for testing, so you don't need actual VPS connection to see it working.
